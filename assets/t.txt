Папка api/

openrouter.py - клиент для работы с API OpenRouter.ai. Его задача отвечать за всю коммуникацию с сервером. Содержит методы для отправки сообщений и получения ответов.
Папка ui/

components.py - UI компоненты (пузырьки сообщений, селектор моделей).
styles.py - все стили приложения в одном месте. Отвечает за визуальную часть приложения.
Папка utils/

analytics.py - сбор статистики использования.
cache.py - кэширование диалогов в SQLite.
logger.py - система логирования.
monitor.py - мониторинг производительности.
Корневые файлы

.env - конфигурация и API ключи.
main.py - точка входа в приложение.
main_simple.py - точка входа для упрощенной версии приложения.
build.py - файл сборки приложения.
requirements.txt - зависимости проекта.
